# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue in this repository, please do not open a public issue.

Use GitHub's private security advisory flow for the repository if available, or contact the maintainers directly through a private channel.

Please include:
- A clear description of the issue
- Steps to reproduce
- Any affected versions or devices
- Screenshots, logs, or sample inputs if they help

We will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
